\doxysubsection{Detailed Description}
RAII wrapper for C FILE\texorpdfstring{$\ast$}{*} operations. 

This class provides automatic resource management for file operations. It wraps standard C file functions (fopen, fclose, fread, fwrite, etc.) with RAII semantics and move support.\hypertarget{classufo_1_1ufile__t_example_usage}{}\doxysubsection{\texorpdfstring{Example Usage}{Example Usage}}\label{classufo_1_1ufile__t_example_usage}

\hypertarget{classufo_1_1ufile__t_thread_safety}{}\doxysubsection{\texorpdfstring{Thread Safety}{Thread Safety}}\label{classufo_1_1ufile__t_thread_safety}

\begin{DoxyItemize}
\item Different instances for different files are safe to use from different Free\+RTOS tasks
\item One instance is NOT safe to use from different tasks without external synchronization (mutex)
\item Individual method calls (read\+\_\+rb/write) are atomic at the VFS level, but the sequence of operations requires synchronization
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
Not thread-\/safe for same instance across multiple tasks
\end{DoxyWarning}

All methods return a bool to indicate success/failure. Detailed error information can be obtained via \doxylink{classufo_1_1ufile__t_aea7b26d5c11ec20710c64ebe04f17f18}{last\+\_\+error()}
\end{DoxyNote}
