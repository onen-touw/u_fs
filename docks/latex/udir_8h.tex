Directory operations class for ESP32 file systems.

Provides comprehensive directory and file operations for different file systems on ESP32 (SPIFFS, Little\+FS, FAT). Handles FS-\/specific features like SPIFFS flat structure.

\begin{DoxyNote}{Note}
SPIFFS Limitations\+:
\begin{DoxyItemize}
\item No real directories (emulated via prefixes)
\item create\+\_\+subdir() and remove\+\_\+subdir() return false
\item list\+\_\+subdirs() always returns empty
\end{DoxyItemize}
