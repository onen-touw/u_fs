Provides an interface for operations on files and subdirectories taking into account the features of different file systems (SPIFFS, Little\+FS, FAT).

\begin{DoxyNote}{Note}
Features\+:
\begin{DoxyItemize}
\item SPIFFS\+: flat structure, directories are emulated through prefixes
\item Little\+FS/\+FAT\+: full directory hierarchy
\item Does not support copying and moving between different file systems
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
The class is not thread safe. For use from different tasks require external synchronization.
\end{DoxyWarning}
