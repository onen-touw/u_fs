Provides comprehensive directory and file operations for different file systems on ESP32 (SPIFFS, LittleFS, FAT). Handles FS-specific features like SPIFFS flat structure.

Note
SPIFFS Limitations:
  • No real directories (emulated via prefixes)
  • create_subdir() and remove_subdir() return false
  • list_subdirs() always returns empty
